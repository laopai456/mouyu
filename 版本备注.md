### 项目版本备注

**版本号**：v1.0.3  
**发布日期**：2026-03-25  
**版本类型**：功能更新和问题修复

#### 主要功能
1. **Web 管理工具**
   - 批量上传图片（无数量限制）
   - 图片审核（通过/拒绝）
   - 批量删除图片
   - 数据统计（待审核/已通过/已拒绝）
   - 图片预览模态框
   - 按月份筛选查看图片

2. **权限管理**
   - 开发者：无限上传次数
   - 共创者：每日有限上传次数
   - 普通用户：无上传权限

3. **图片管理**
   - 重复图片检测（MD5）
   - 临时 URL 获取
   - 图片预览优化
   - 按月份自动分类存储
   - 自动清理 3 天前的图片

#### 技术改进
- 升级 Web SDK 到最新版本
- 实现云函数获取临时 URL
- 实现云函数批量删除
- 实现云函数自动清理
- 配置云函数安全规则
- 修复 iOS 设备图片加载问题
- 修复基础库兼容性问题
- 添加日期字段支持按月份管理

#### 云开发配置要求
- 开启「未登录用户访问云资源权限」
- 开启「允许匿名登入」
- 数据库权限：images 集合设置为「所有用户可读，创建者可写」
- 存储权限：设置为「所有用户可读，创建者可写」
- 云函数权限：
  - getTempUrls: `{"*": {"invoke": true}}`
  - deleteImages: `{"*": {"invoke": true}}`
  - autoCleanup: 配置定时触发器 `0 0 2 * * * *`

#### 已知问题
- 临时 URL 有 2 小时有效期，过期后需重新获取
- 免费版云存储权限设置有限制

#### 部署步骤
1. 部署云函数：
   - getTempUrls
   - deleteImages
   - autoCleanup
2. 配置云函数权限
3. 配置定时触发器（autoCleanup）
4. 启动本地服务器访问 Web 管理工具
