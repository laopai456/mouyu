# Admin 批量审核功能待办

## 功能需求

### 待审核页面改造

**8张图网格预览模式**：
- 同时显示8张待审核图片（4列2行网格）
- 每张图左上角有勾选框，默认不勾选
- 右下角两个操作按钮

**操作按钮**：
| 按钮 | 功能 |
|------|------|
| 一键拒绝 | 拒绝所有未勾选的图片（默认全部拒绝） |
| 勾选通过 | 勾选的通过审核，未勾选的拒绝 |

**翻页逻辑**：
- 处理完当前批次后，自动加载下一批8张

**模式切换**：
- 保留原有单张预览模式
- 添加切换按钮，在两种模式间切换

## 使用说明

1. 进入管理后台 → 待审核 Tab
2. 点击「网格审核」按钮切换到网格模式
3. 点击图片左上角的勾选框选择要通过的图片
4. 点击图片可以放大预览
5. 操作完成后点击底部按钮：
   - 「一键拒绝未勾选」：拒绝所有未勾选的图片
   - 「勾选通过，未勾选拒绝」：勾选的通过，未勾选的拒绝
6. 处理完成后自动加载下一批
